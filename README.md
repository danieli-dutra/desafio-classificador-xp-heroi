# 🦸 Classificador de Nível de Herói

Este projeto foi desenvolvido como parte de um desafio prático da DIO, com o objetivo de fortalecer minha base em lógica de programação utilizando JavaScript.

Durante o desenvolvimento, aproveitei este desafio para praticar organização de código, entrada de dados via terminal e construção de regras de decisão através de estruturas condicionais.

## 🎯 Objetivo do desafio

Criar um sistema capaz de:

- Receber o nome de um herói
- Receber sua quantidade de XP
- Classificar automaticamente seu nível com base em faixas de experiência

## 🧠 Minha abordagem

Para resolver este desafio, trabalhei com:

- Declaração de variáveis
- Conversão de tipos de dados
- Estruturas condicionais (`if`, `else if`, `else`)
- Entrada de dados pelo terminal com `prompt-sync`
- Organização da lógica pensando em futuras evoluções

## 🚀 Tecnologias utilizadas

- JavaScript
- Node.js
- prompt-sync

## 📚 Aprendizados durante o desenvolvimento

Durante este projeto, pratiquei conceitos importantes como:

✔ Diferença entre ambiente navegador e Node.js  
✔ Tratamento de entrada de dados pelo terminal  
✔ Estruturação de regras de negócio  
✔ Debug e resolução de erros de sintaxe  
✔ Organização inicial de projeto para GitHub  


## ▶ Como executar

```bash
git clone https://github.com/danieli-dutra/desafio-classificador-xp-heroi.git
cd desafio-classificador-xp-heroi
npm install
node src/classificacaoHeroi.js
```

## 💻 Exemplo de uso

```bash
Qual é o nome do herói? Peter
Qual é a quantidade de XP do herói? 8500

O Herói de nome Danieli está no nível de Ascendente
```
